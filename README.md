# Exploratory Data Analysis (EDA) Project

## Overview

This project performs EDA on a student dataset to find patterns and relationships between study hours, attendance, age, and marks.

## Objectives

* Analyze the dataset
* Generate statistics
* Visualize data
* Find correlations
* Create an EDA report

## Dataset

File: `students.csv`

### Features

* Student_ID
* Age
* Gender
* Study_Hours
* Attendance
* Marks

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Installation

```bash
pip install pandas numpy matplotlib seaborn
```

## Project Structure

```text
EDA_Project/
├── eda_project.py
├── students.csv
├── README.md
└── Output Files
```

## How to Run

```bash
python eda_project.py
```

## Outputs

* EDA_Report.txt
* Histograms
* Correlation Heatmap
* Pair Plot
* Boxplots

## Insights

* Study Hours vs Marks
* Attendance vs Marks
* Performance Distribution
* Outlier Detection

## Conclusion

This project uses EDA techniques to analyze student data and generate useful insights.
